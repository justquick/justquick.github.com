Running the 12step commands¶

12step-init¶

Iniitialize your custom configuration interactively by answering a few questions

usage: 12step-init [-h] [-o OUTPUT]
-h, --help¶

show this help message and exit

-o <output>, --output <output>¶

Filename to output your config to

12step-download¶

Download meeting data from a WordPress site with 12-step-meeting-list plugin.

usage: 12step-download [-h] [--config CONFIG] [--verbose] [--logfile LOGFILE]
                       [-u SITE_URL] [-f {json,csv}] [-s SECTIONS]
-h, --help¶

show this help message and exit

--config <config>, -c <config>¶

Configs file for runtime vars. Can pass multiple to override options.

--verbose, -v¶

More verbose logging

--logfile <logfile>¶

Optional log file to wrie to

-u <site_url>, --site-url <site_url>¶

WordPress Site URL root

-f {json,csv}, --format {json,csv}¶

Format of downloaded meeting data

-s <sections>, --sections <sections>¶

Comma separated list of sections to download

12step-pdf¶

Format meeting PDF

usage: 12step-pdf [-h] [--config CONFIG] [--verbose] [--logfile LOGFILE]
                  [--output OUTPUT] [--download] [--limit LIMIT]
-h, --help¶

show this help message and exit

--config <config>, -c <config>¶

Configs file for runtime vars. Can pass multiple to override options.

--verbose, -v¶

More verbose logging

--logfile <logfile>¶

Optional log file to wrie to

--output <output>, -o <output>¶

Output file name to render to. Use “-” for stdout

--download, -d¶

Download the assets before rendering. Produces up to date PDFs

--limit <limit>, -l <limit>¶

Limit the rendering to this number of meetings

12step-flask¶

Run Flask webapp for Development. This should never be run in production! You must restart this if you make any changes to code or asset files

usage: 12step-flask [-h] [-a ADDRESS] [-p PORT]
-h, --help¶

show this help message and exit

-a <address>, --address <address>¶

The host interface address to bind to

-p <port>, --port <port>¶

The port to bind to

Generates PDFs meeting guides from sites using the 12 Step Meeting List WordPress plugin
